# qxis.site

This is a personal portfolio website built with [Next.js](https://nextjs.org), featuring a custom glassmorphism design, SVG emojis, and Next.js static asset optimizations for GitHub Pages.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploying

Deploy to GitHub pages automatically using the included script:
```bash
npm run deploy
```
